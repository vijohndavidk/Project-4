# **Netflix Clone Website**

A responsive Netflix Clone Website built using HTML and CSS.  
This project recreates the homepage UI of Netflix with a modern dark theme, hero banner, trending movies section, hover effects, and responsive design.

---

# **Features**

* Netflix-style homepage  
* Responsive navigation bar  
* Hero banner section  
* Email signup form  
* Trending movie posters  
* Hover zoom animation  
* Mobile responsive layout  
* Dark theme design

---

# **Technologies Used**

* HTML5  
* CSS3  
* VS Code  
* Live Server
